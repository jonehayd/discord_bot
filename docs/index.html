Economy

6 commands
/economy balance

Check a user's 🥥 coconut balance.

Options
user The user to check (defaults to you) optional
/economy leaderboard

Displays the top 5 users with the most 🥥 coconuts.

/economy addbalance

Add coconuts to a user's balance.

Options
user The user to add money to required
amount The amount to add (min: 1) required
/economy removebalance

Remove coconuts from a user's balance.

Options
user The user to remove money from required
amount The amount to remove (min: 1) required
/economy resetbalance

Reset a user's balance back to zero.

Options
user The user whose balance will be reset required
/economy transferbalance

Transfer coconuts from one user to another.

Options
user_from The user to take money from required
user_to The user to transfer money to required
amount The amount to transfer required

Fun

7 commands
/fun poll

Start a yes/no poll with reaction voting. Results are posted when time expires.

Options
question The poll question required
duration How long the poll runs in minutes (1–60, default: 1) optional
/fun rob

Attempt to steal 🥥 100 coconuts from another user. They must confirm or deny within a time window. Has a 10-minute cooldown.

Options
target The user you want to rob required
/fun trivia

Answer a trivia question to win 🥥 coconuts! Rewards scale with difficulty: easy 50, medium 100, hard 200.

Options
difficulty easy · medium · hard (default: medium) optional
category General · Film · Music · TV · Video Games · Science · Computers · Sports · Geography · History optional
/fun roast

AI generates a savage, comedic roast of a target user using their profile, recent channel messages, and Steam library (if linked). 30-second cooldown.

Options
target The user to roast required
/fun compliment

AI generates an enthusiastic, personalised compliment for a user based on their profile, recent messages, and Steam library (if linked). 30-second cooldown.

Options
target The user to compliment required
/fun wouldyourather

AI generates a "Would You Rather" dilemma for the server to vote on. Buttons track live votes for 5 minutes, then lock and show the final result.

Options
theme Optional topic for the question, e.g. "food", "superpowers" optional
/fun chat

Send any message to the AI and get a response posted in the channel. 10-second cooldown. Max 1000 characters.

Options
message What do you want to say? required

Music

13 commands
/music play

Play a song from YouTube by name or URL. Joins your voice channel automatically.

Options
query Song name or YouTube URL required
/music pause

Pause the currently playing song.

/music resume

Resume paused playback.

/music skip

Skip the currently playing song and advance to the next in the queue.

/music stop

Stop playback, clear the entire queue, and disconnect the bot from the voice channel.

/music queue

Show the current music queue along with the now-playing track.

/music nowplaying

Display information about the currently playing song.

/music loop

Toggle looping the current song. When enabled, the song repeats indefinitely.

/music shuffle

Randomly shuffle all songs in the current queue.

/music remove

Remove a specific song from the queue by its position number.

Options
position Queue position to remove (min: 1). Use /music queue to see positions. required
/music autoplay

Toggle autoplay — when the queue is empty, the bot continues with YouTube recommendations.

/music ctrl

Open an interactive music control panel with buttons for play, pause, skip, and more.

Utility

6 commands
/utility ping

Check if the bot is responsive. Replies with "Pong!" and the current latency.

/utility choose

Randomly pick one option from a comma-separated list. Great for making decisions.

Options

/utility remind

Set a reminder that pings you after a specified time (max 7 days).

Options
message What to remind you about required
hours Hours to wait (0-168) optional
minutes Minutes to wait (0-59) optional
seconds Seconds to wait (0-59) optional
visibility Where to confirm the reminder — public or private (default: public) optional
/utility server

Display information about the current server: name, member count, creation date, and more.

/utility user

Display information about a user: username, avatar, account creation date, and server join date.

/utility help

List all available bot commands and how to use them, right inside Discord.
